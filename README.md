### Setup
  ```bash
  git clone https://github.com/AndriiBuhalo/time_getter.git
  cd time_getter
  ```

before first start
1. bundle install
2. rails db:create db:migrate

### Run the servers locally
``` rails s```

default port 3000

### POSTMAN COLLECTION
You can find inside file:
```./spec/support/TimeFinder.postman_collection.json```
